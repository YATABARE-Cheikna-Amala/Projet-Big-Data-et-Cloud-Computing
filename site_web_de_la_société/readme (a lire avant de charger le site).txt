## Mise en place de l'application (vous devez d'abord installer Xamp)

1- Mettez ce dossier (laplaceImmo) dans le dossier htdocs qui se situe dans le dossier xamp créé lors de l'installation.

2- Lancez le serveur xamp (apache et mysql)

3- Tout est ok maintenant, tapez sur votre navigateur : localhost/laplaceImmo/index.html
